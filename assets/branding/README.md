# Federated Group Branding Assets

This directory contains Federated Group branding assets including logos and brand materials.

## Required Logo Files

Please place the following logo files in this directory:

### Main Logo
- **File**: `logo.png`
- **Format**: PNG with transparent background
- **Minimum Size**: 200px width
- **Usage**: Primary logo for most applications

### White Logo
- **File**: `logo_white.png`
- **Format**: PNG with transparent background
- **Usage**: For dark backgrounds

### Black Logo
- **File**: `logo_black.png`
- **Format**: PNG with transparent background
- **Usage**: For light backgrounds

### Icon Logo
- **File**: `logo_icon.png`
- **Format**: PNG
- **Size**: 64x64px or 128x128px
- **Usage**: Favicons, small icons, mobile apps

### Horizontal Logo
- **File**: `logo_horizontal.png`
- **Format**: PNG with transparent background
- **Usage**: Wide headers, banners, email signatures

## Logo Specifications

### File Formats
- **Primary**: PNG (with transparency)
- **Alternative**: SVG (if available, for scalability)

### Size Requirements
- **Main Logo**: Minimum 200px width, high resolution recommended
- **Icon**: 64x64px or 128x128px
- **Horizontal**: Appropriate width for banner use

### Quality Requirements
- High resolution (300 DPI for print materials)
- Clear and crisp at all sizes
- Proper transparency (no white background artifacts)

## Obtaining Logo Files

Contact the following for logo files:
- Marketing/Branding Team
- Design Team
- Corporate Communications

## Usage

Once logo files are placed in this directory, they will be automatically used in:
- Power BI dashboards
- API documentation
- Generated reports
- Email templates
- Documentation headers

See `docs/BRANDING_GUIDE.md` for detailed usage guidelines.

## Notes

- Logo files are not included in the package by default
- Add logo files before deployment
- Ensure logo files are properly licensed for use
- Maintain brand guidelines when using logos
